# Products Management System

## Description
This project is a simple Products Management System built using PHP. It includes functionalities for managing users, products, and authentication.

## Features
- User authentication (login/logout)
- Add new products via a modal form
- Manage users

## File Structure
- **db.php**: Handles database connection.
- **index.php**: Main entry point of the application.
- **login.php**: Handles user login functionality.
- **logout.php**: Handles user logout functionality.
- **modal_add.php**: Contains the modal form for adding new products.
- **users.php**: Manages user-related operations.

## Requirements
- PHP 7.4 or higher
- MySQL database
- A web server (e.g., Apache, Nginx)

## Installation
1. Clone the repository to your local machine:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd products
   ```
3. Set up the database:
   - Import the provided SQL file (if available) into your MySQL database.
   - Update the database credentials in `db.php`.
4. Start your web server and access the application via your browser.

## Usage
1. Open the application in your browser.
2. Log in using your credentials.
3. Use the interface to manage products and users.
